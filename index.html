Robot Overview

Platform, drive, sensors, and mechanism

Hover a component

Explore the EV3 robot blueprint to learn how each subsystem supports Rescue Line obstacles.

  • Platform: LEGO Mindstorms EV3 (Education edition)
  • Drive: 2 Large Motors with grippy wheels + gyro leveling
  • Sensors: Dual color, ultrasonic range, Pixy2 vision cam for obstacle cues
  • Mechanism: Custom grabber powered by EV3 Medium Motor for victim rescue

Technical Features

Advanced Line Following

Dual color sensor system for robust acquisition and tracking of the black line with white boundary detection for reliability under lighting changes.

Obstacle Management

Pixy2 camera flags obstacles by color/size while the ultrasonic sensor validates range, letting the EV3 reroute smoothly.

Autonomous Navigation

Handles intersections, gaps, inclines, and checkpoints per 2025 rules with state-based logic and recovery behaviors.

Custom Engineering

Innovative sensor placement and an optimized grabber designed for precise, quick evacuation zone rescues.

Field Operations

01

Line Following

Weighted sensor fusion adjusts motor speeds to center on the line while respecting the white boundary.

02

Obstacle Avoidance

Pixy2 obstacle cues escalate to ultrasonic confirmation; when range closes, the EV3 executes a bypass arc and re-acquires the line.

03

Rescue Sequence

In evacuation zone, the grabber aligns, secures the “victim,” and transports to the safe area.

Autonomous Run

  1. Initialize sensors, calibrate light thresholds, and set base motor speed.
  2. Follow line using dual-sensor PID with boundary validation.
  3. Handle intersections: decide via priority rules, continue tracking.
  4. Bridge gaps: predictive search and incremental sweep to re-locate line.
  5. Inclines: increase torque and maintain sensor height consistency.
  6. Pixy2 scans for obstacles/victims; successful detections gate ultrasonic confirmation before maneuvers.
  7. Enter evacuation zone, detect target, actuate grabber, deliver to safe zone.
